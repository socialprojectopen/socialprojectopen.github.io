# Сайт UBS Transit

Многостраничный сайт компании UBS Transit для доставки сборных грузов по России и странам СНГ.

## Описание

Сайт создан на основе анализа дизайна tek-kub.ru и содержит информацию о компании ЗАО «ЮуБиЭс Транзит», специализирующейся на доставке сборных грузов автотранспортом.

## Структура сайта

### Страницы

1. **index.html** - Главная страница
   - Hero секция с заголовком
   - Форма расчета стоимости доставки
   - Описание услуг
   - Краткие преимущества
   - Призыв к действию

2. **about.html** - О компании
   - Полная информация о компании
   - Специализация на сборных грузах
   - География доставки
   - Реквизиты компании
   - Информация о директоре

3. **reviews.html** - Отзывы
   - Отзывы клиентов компании
   - Карточки с отзывами
   - Призыв стать клиентом

4. **advantages.html** - Преимущества
   - Детальное описание 8 преимуществ
   - Преимущества сборных грузов
   - Призыв к действию

5. **contact.html** - Контакты
   - Контактная информация
   - Адреса (юридический, почтовый, на английском)
   - Форма обратной связи
   - Реквизиты для оплаты

## Использование

1. Откройте `index.html` в браузере для просмотра главной страницы
2. Навигация между страницами осуществляется через меню в шапке сайта
3. Все страницы имеют единый стиль и структуру
