# План создания многостраничного сайта UBS Transit

- **Юридический адрес**: Кыргызская Республика, Чуйская область, Московский район, село АкСуу, ул. Дружбы, 16B, 720011

## Тематика сайта
- Доставка сборных грузов по России и странам СНГ
- Только автотранспортом
- Междугородние перевозки

## Структура сайта (многостраничный)

### 1. Главная страница (index.html)
- Hero секция с заголовком о доставке сборных грузов
- Форма расчета стоимости доставки
- Краткое описание услуг
- Преимущества (кратко)
- Призыв к действию

### 2. О компании (about.html)
- Полная информация о компании
- История и опыт
- Специализация (сборные грузы, автотранспорт)
- Реквизиты компании
- Директор

### 3. Отзывы (reviews.html)
- Отзывы клиентов
- Карточки с отзывами
- Рейтинг/статистика

### 4. Преимущества (advantages.html)
- Детальное описание преимуществ
- Почему выбирают нас
- Преимущества сборных грузов

### 5. Контакты (contact.html)
- Контактная информация
- Адреса
- Форма обратной связи
- Карта (если возможно)

## Технические требования

### Дизайн
- Использовать цветовую палитру из анализа tek-kub.ru
- Шрифты: Roboto (основной), Montserrat (заголовки)
- Минималистичный дизайн
- Без PNG, SVG, эмодзи
- Без стилей нейросетей

### Структура файлов
```
/
├── index.html
├── about.html
├── reviews.html
├── advantages.html
├── contact.html
├── css/
│   ├── style.css (основные стили)
│   └── normalize.css (сброс стилей)
└── js/
    └── main.js (если нужен)
```

### Общие элементы
- Header с навигацией (одинаковый на всех страницах)
- Footer с контактами (одинаковый на всех страницах)
- Единый стиль карточек
- Единый стиль кнопок
- Адаптивный дизайн
