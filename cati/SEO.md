## Структура файлов

```
cati/
├── sitemap.xml          # Карта сайта для поисковых систем
├── robots.txt           # Правила индексации
├── index.html           # Главная страница (с полным набором Schema.org)
├── products.html        # Каталог (с BreadcrumbList и CollectionPage)
├── news.html           # Новости (с Blog schema)
└── certificates.html   # Сертификаты (с BreadcrumbList)
```

## Рекомендации для дальнейшего улучшения

### 1. Google Search Console
- Зарегистрировать сайт в Google Search Console
- Добавить и подтвердить sitemap.xml
- Отслеживать индексацию и ошибки

### 2. Яндекс.Вебмастер
- Зарегистрировать сайт в Яндекс.Вебмастере
- Добавить sitemap.xml
- Настроить регион и тематику сайта

### 3. Дополнительные улучшения
- Добавить favicon в разных размерах (16x16, 32x32, 192x192, 512x512)
- Оптимизировать изображения (WebP формат, lazy loading уже реализован)
- Добавить микроразметку для статей новостей (Article schema)
- Реализовать хлебные крошки визуально на страницах
- Добавить alt-тексты для всех изображений (уже частично реализовано)

### 4. Производительность
- Минифицировать CSS и JS файлы
- Включить GZIP сжатие на сервере
- Настроить кэширование статических ресурсов

## Проверка SEO

### Инструменты для проверки:
1. **Google Rich Results Test** - https://search.google.com/test/rich-results
   - Проверка структурированных данных Schema.org

2. **Google Mobile-Friendly Test** - https://search.google.com/test/mobile-friendly
   - Проверка мобильной адаптивности

3. **PageSpeed Insights** - https://pagespeed.web.dev/
   - Проверка скорости загрузки

4. **Yandex Webmaster Tools** - https://webmaster.yandex.ru/
   - Проверка индексации в Яндексе

5. **Facebook Sharing Debugger** - https://developers.facebook.com/tools/debug/
   - Проверка Open Graph тегов

6. **Twitter Card Validator** - https://cards-dev.twitter.com/validator
   - Проверка Twitter Cards
