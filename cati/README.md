# Сайт ООО «ЦАТИ»

Профессиональный корпоративный сайт Центра Авиационных Технологий и Интерьера.

## Структура проекта

```
cati-new/
├── index.html          # Главная страница
├── products.html       # Каталог продукции
├── certificates.html   # Лицензии и сертификаты
├── news.html          # Новости/Блог
├── css/
│   └── style.css      # Основные стили (mobile-first)
├── js/
│   └── main.js        # JavaScript для навигации и интерактивности
├── images/            # Изображения
└── pdf/               # PDF файлы (сертификаты и лицензии)
```

## Особенности

- Профессиональный деловой дизайн
- Полная мобильная адаптация (mobile-first)
- Структурированная навигация
- Адаптивная типографика
- Оптимизация производительности
- Семантическая разметка HTML5
- Доступность (ARIA атрибуты)

## Технологии

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Mobile-first responsive design

## Локальный запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
cd cati-new
python -m http.server 8000
```

Затем откройте http://localhost:8000 в браузере.

## Страницы сайта

1. **Главная страница** (`index.html`) - О компании, услуги, преимущества, контакты
2. **Каталог** (`products.html`) - Описание продукции по направлениям
3. **Сертификаты** (`certificates.html`) - Лицензии и сертификаты с PDF файлами
4. **Новости** (`news.html`) - Блог компании
