# Сайт ЦВ-Агент

Сайт компании "ЦВ" (Цифровая Валюта) - агент для международных платежей и переводов.

## Описание

Сайт создан на основе дизайна A7-Agent.ru с адаптацией под компанию ЦВ-Агент. Включает все основные функции оригинального сайта, кроме регистрации/входа в личный кабинет.

## Структура проекта

```
├── index.html          # Главная страница
├── styles.css          # Основные стили
├── design_tokens.css   # Дизайн-токены (цвета, типографика)
├── script.js           # JavaScript для интерактивности
└── README.md           # Документация
```

## Особенности

### Дизайн
- Современный, минималистичный дизайн
- Цветовая схема: оранжевый (#F88F35) для акцентов, бирюзовый (#219096) для доверия
- Типографика: Raleway для заголовков, Inter для текста
- Адаптивная верстка для всех устройств

### Функциональность
- ✅ Калькулятор стоимости перевода
- ✅ Формы заявок
- ✅ FAQ с аккордеоном
- ✅ Плавная прокрутка
- ✅ Cookie уведомление
- ❌ Регистрация/вход (не реализовано по требованию)

### Секции сайта

1. **Header** - Шапка с логотипом и контактами
2. **Banner** - Главный баннер с призывом к действию
3. **Преимущества** - Карточки с преимуществами компании
4. **Калькулятор** - Расчет стоимости перевода
5. **Как мы работаем** - Пошаговый процесс работы
6. **Форма заявки** - Форма для связи
7. **FAQ** - Часто задаваемые вопросы
8. **Footer** - Подвал с контактами и правовой информацией

## Использование

1. Откройте `index.html` в браузере
2. Все стили и скрипты подключены автоматически
3. Для разработки можно использовать локальный сервер:
   ```bash
   python -m http.server 8000
   ```
   Затем откройте http://localhost:8000

## Настройка

### Изменение контактов
Отредактируйте в `index.html`:
- Телефон: `<a href="tel:+78001234567">8-800-123-45-67</a>`
- Email: `<a href="mailto:info@cv-agent.ru">info@cv-agent.ru</a>`

### Изменение цветов
Отредактируйте переменные в `design_tokens.css`:
```css
--color-primary: #F88F35;        /* Основной акцент */
--color-secondary: #219096;      /* Вторичный акцент */
```

### Изменение курсов валют
Отредактируйте в `script.js`:
```javascript
const exchangeRates = {
    'USD': 78.5,
    'EUR': 85.2,
    'CNY': 10.8,
    'AED': 21.4
};
```
