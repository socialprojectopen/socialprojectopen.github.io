# Детальный анализ структуры сайта Оклейка.рф

**URL:** https://xn--80ajmccdt.xn--p1ai/ (оклейка.рф)  
**Тема:** Брендирование и реклама на автомобиль

---

## 1. Технологический стек

- **CMS:** WordPress 6.9
- **Тема:** Roof (VictorThemes) v2.1
- **Page builder:** Elementor 3.34.1
- **Плагины:** Header Footer Elementor, Revolution Slider, Contact Form 7, WPBakery (JS Composer), Max Mega Menu, Popup Maker, Chaty, TablePress, Easy Fancybox, Color Palette и др.

## 2. Структура страницы (DOM / секции)

1. **Cookie-баннер** — модальное окно «Мы используем cookie» с кнопкой «Согласен, принимаю»

   - Слайдер (Revolution Slider) — несколько слайдов с CTA
   - Блок с заголовком H1 «Брендирование и реклама на автомобиль»
   - Секция **«НАШИ УСЛУГИ»** (H2) — сетка карточек услуг (оклейка легковых, Ларгус, фургоны, заднее стекло, ГАЗель, грузовики, фуры, тент)
   - Текстовый блок «О компании» (с 2008 года, преимущества)
   - Секция **«ДОПОЛНИТЕЛЬНЫЕ УСЛУГИ»** (H2) — печать на пленке, таблички ПВХ, наклейки, дизайн
   - Секция **«ПРЕИМУЩЕСТВА»** (H2) — иконки + текст (опыт, производство, качество, бригады, бокс, 24/7, гарантия)
   - Форма **«ПОЛУЧИТЬ КОНСУЛЬТАЦИЮ»** (Contact Form 7): имя, email, телефон, комментарии, прикрепление файла
   - Секция **«НАШИ РАБОТЫ»** (H2) — галерея изображений, кнопка «Посмотреть еще»
   - Секция **«НАШИ ПАРТНЕРЫ»** (H2)
   - Карта (iframe Яндекс.Карты)
4. **Подвал (footer/contentinfo):**
   - «Работаем с 2008 года», навигация по услугам (многоуровневое меню), контакты, форма «Заказать звонок», копирайт © 2020 Оклейка.рф

## 3. Стили и оформление

### 3.1 Цветовая палитра (из темы Roof)

| Назначение        | Значение        | Описание              |
|-------------------|-----------------|------------------------|
| Основной акцент   | `#eeb313`       | Золотисто-оранжевый    |

### 3.2 Типографика (шрифты)

**Основные шрифты на сайте:**

- **Raleway** — основной для body, заголовков, меню, кнопок (Google Fonts: 400, 500, 600, 700)
- **Open Sans** — параграфы, формы, контактная информация (400, 500, 700)
- **Roboto** — части заголовков, кнопки «большие» (100–900, latin + cyrillic)
- **Roboto Slab** — акцентные заголовки, команда (serif)
- **Russo One** — акцентные заголовки (например, в слайдере)
- **Montserrat** — местами в виджетах
- **Helvetica Neue / Arial** — фолбэк
- **Font Awesome 5 Free** — иконки (в т.ч. в шапке/меню)
- **Elementor Icons (eicons)** — иконки в блоках Elementor

**Размеры (из theme styles):**

- Body: `10px` (базовый в теме), цвет `#232323`
- Параграфы: `14px`, `line-height: 25px`, цвет `#888888`
- H1: ~`24px` (на главной), в слайдере крупнее (40–50px)
- H2: `30px` (секции «НАШИ УСЛУГИ», «ПРЕИМУЩЕСТВА» и т.д.)
- H3–H6: 10px margin-bottom, font-weight 700, line-height 1.43
- Кнопки: `11–13px`, uppercase, letter-spacing 0.2–0.5px
- Меню: `13px`, uppercase, font-weight 700

### 3.3 Ключевые стили элементов

- **Кнопки (.roof-btn):**  
  фон `#eeb313`, цвет текста `#efefef`, при hover `#d59f0e`. Варианты: gray, black, white, orange-border.
- **Формы (input/textarea):**  
  высота 50px, padding 10px 20px, граница `1px solid #ececec`, фон `#ffffff`, шрифт Open Sans 14px.
- **Submit-кнопки:**  
  фон `#eeb313`, цвет текста `#232323`, uppercase, Raleway 700.
- **Ссылки:**

- **Контейнеры:**  
  `.roof-container` 880px, boxed layout до 1272px; контент Elementor в своей сетке.
- **Слайдер:**  
  Swiper, высота 600–700px, фон слайдов `#d0d0d0`, оверлей с rgba(0,0,0,.27–.55). Подпись (.cpation-wrap): фон rgba(0,0,0,.55), подзаголовок цвет `#eeb313`.

## 4. Анимации и ключевые кадры (keyframes)

Используются анимации из:

- **Тема Roof:**  
  `fadeIn` (opacity 0 → 1)
- **Elementor:**  
  `fade-in`, `fade-out`, `slide-in-left`, `slide-in-right`, `slide-out-*`, `components-animate__appear-animation`, `components-animate__slide-in-animation`, `components-animate__loading`, `components-button__busy-animation`, `components-modal__*`, `components-resizable-box__*`, `components-validated-control__indicator-jump`
- **Revolution Slider:**  
  `rev-ani-mouse`, слои слайдов
- **WPBakery:**  
  `wpb_ttb`, `wpb_btt`, `wpb_ltr`, `wpb_rtl`, `wpb_appear`
- **Animate.css (через JS Composer):**  
  `fadeIn`, `fadeOut`, `bounce`, `pulse`, `shake`, `zoomIn`, `zoomOut`, `slideIn*`, `slideOut*`, `rotateIn*`, `flipIn*`, `lightSpeedIn`, `rollIn`, `jackInTheBox` и др.
- **Popup Maker:**  
  `components-modal__appear-animation`, `components-modal__disappear-animation`
- **Lightbox (Fancybox):**  
  `fancybox-loading`
- **Swiper:**  
  `swiper-preloader-spin`
- **Chaty (виджет чата):**  
  `chaty-animation-shockwave`, `blink`, `pulse`, `sheen`, `fade`, `spin`, `bounce`, `waggle`, `floating`
- **Битрикс24 виджет:**  
  `b24-widget-button-visible`, `b24-widget-button-disable`, `b24-widget-show`, `b24-widget-hidden`, `hideIconsBottom`, `hideIconsTop`, `bottomOpen`, `topOpen`, `socialRotate`, `widgetPulse`
- **Font Awesome:**  
  `fa-spin`
- **Тема (progress, loader):**  
  `progress-bar-stripes`, `loadingpulse`, `tp-rotateplane`, `tp-scaleout`, `tp-bounce`, `spin`

Типичные длительности: `0.3s ease` для переходов (hover, focus).

## 5. Элементы наполнения

### 5.1 Слайдер (главный экран)

- Слайды с фоновыми изображениями и текстом поверх.
- Блоки: «Брендирование тента» (акция 30%), «Рассчитайте стоимость за 15 минут», «Реализовано 100+ проектов», «Оклейка транспорта», телефон + слоган, «Лада Ларгус».
- Кнопки: «Узнать подробнее», «Получить предложение», «Подробнее о ларгусах».
- Навигация: стрелки влево/вправо, пагинация (буллеты).

### 5.2 Карточки услуг

- Заголовок (H3) + ссылка, часто с изображением (figure + img).
- Варианты: только заголовок; заголовок + «Бесплатный макет» / «Подробнее..»; акция «АКЦИЯ! СКИДКА 15%» + «Узнать подробнее».
- Сетка: две колонки по 4 и 4 (или аналогичная разбивка) для восьми видов оклейки; ниже — 4 карточки доп. услуг.

### 5.3 Блок «О компании»

- Короткие абзацы с strong: год основания, лидер рынка, полный цикл (индивидуальный проект, реализация), преимущества (дизайн без шаблонов, поддержка).

### 5.4 Преимущества

- Иконка + заголовок/цифра + описание в одну-две строки (опыт, производство, качество печати, бригады 3M, теплый бокс, 24/7, гарантия).

### 5.5 Форма консультации

- Поля: Имя (placeholder «Иван Иванов»), Email («ivan@example.com»), Телефон («+7 (900) 000-00-00»), Комментарии («Расскажите о проекте...»), прикрепление файла (эскиз/фото авто).
- Кнопка: «Отправить заявку».

### 5.6 Галерея и партнеры

- «Наши работы» — изображения в сетке, ссылка «Посмотреть еще» на /foto_video/foto/.
- «Наши партнеры» — блок под галереей (логотипы/названия).

### 5.7 Подвал

- Текст «Работаем для Вас с 2008 года! Качество проверенное временем.»
- Меню: Оклейка транспорта (5 подпунктов), Брендирование тента (4 подпункта), Широкоформатная печать (3 подпункта).
- Контакты: адрес (Дмитровское шоссе, 100, стр.2, БЦ «Норд Хаус»), телефон, email.
- Форма «Заказать звонок» (поле телефона + кнопка).
- Копирайт: «Все права защищены © 2020 Оклейка.рф».

### 5.8 Внешние виджеты

- Cookie-уведомление (в начале страницы).
- Карта: iframe Яндекс.Карты (Оклейка.рф, кнопки Пробки, масштаб, «Открыть в Яндекс Картах»).
- Возможны виджеты Chaty и Битрикс24 (по наличию скриптов/стилей).
