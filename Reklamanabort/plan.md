# План создания сайта UBS Transit

## Концепция
Сайт компании ОсОО «ЮуБиЭс Транзит» — услуги по **размещению рекламы на бортах наших автомобилей**, занятых автодоставкой сборных грузов по России и СНГ. Без блоков про акции/скидки.

---

## Этапы

### 1. Подготовка
- [x] План и структура проекта
- [x] Данные предприятия из `карточка.txt`
- [x] Тарифы: аренда борта, печать наклеек, Газели, готовые пакеты

### 2. Дизайн-система
- **Шрифты:** Raleway (заголовки), Open Sans (текст) — Google Fonts
- **Цвета:** основной акцент, тёмный текст, белый фон
- **Сетка:** адаптивная, брейкпоинты под мобильный / планшет / ПК

### 3. Структура сайта
- **Главная** (`index.html`) — герой, суть услуги, CTA
- **О нас** (`about.html`) — компания, реквизиты
- **Услуги** (`services.html`) — тарифы, таблицы цен
- **Контакты** (`contacts.html`) — адрес, телефон, форма

### 4. Общие блоки
- Шапка: логотип/название, навигация (Главная, О нас, Услуги, Контакты), телефон
- Подвал: контакты, копирайт
- Мобильное меню: бургер, выезжающая панель

### 5. Адаптивность
- Mobile first или единая сетка с медиазапросами
- Меню-бургер на узком экране
- Таблицы цен: прокрутка или карточки на мобильном

### 6. Контент
- Тексты без акций
- Таблицы цен из утверждённых тарифов
- Реквизиты и контакты из карточки предприятия
